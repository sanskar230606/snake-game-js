# Snake Game

A classic snake game built with vanilla JavaScript, HTML, and CSS.

## Features

- 🎮 Classic snake gameplay
- 📊 Score tracking with high score persistence using localStorage
- ⏱️ Real-time game timer
- 🎯 Responsive grid-based board
- 🎨 Dark themed UI with smooth animations
- 🏁 Start and game-over modals

## How to Play

1. Click **"Start Game"** to begin
2. Use **Arrow Keys** to control the snake:
   - ⬆️ Up
   - ⬇️ Down
   - ⬅️ Left
   - ➡️ Right
3. Eat the **green food** to grow and earn points (+10 per food)
4. Avoid hitting the **walls** or yourself
5. Try to beat your **high score**!

## Game Mechanics

- Snake starts with 1 segment
- Each food eaten adds 1 segment to the snake
- High score is saved locally in your browser
- Game speed: 200ms per move
- Grid: 50x50px blocks

## Files

- `index.html` - Game structure
- `script.js` - Game logic and mechanics
- `style.css` - Styling (compiled from SCSS)
- `android-chrome-192x192.png` - Favicon
